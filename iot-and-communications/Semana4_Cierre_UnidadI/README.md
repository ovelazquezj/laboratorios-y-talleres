# Evaluación Semana 4 – Modulación FSK
Curso: Comunicaciones e IoT

Este ejercicio evalúa tu comprensión del proceso de modulación digital FSK. Deberás implementar el algoritmo de modulación FSK paso a paso, y validar tu señal usando el script de demodulación incluido.


## PASOS PARA CREAR EL ENTORNO CONDA

1. Abre tu terminal o Anaconda Prompt.
2. Crea un nuevo entorno llamado `iot_modulacion`:
    ```
    conda create -n iot_modulacion python=3.11
    ```
3. Activa el entorno:
    ```
    conda activate iot_modulacion
    ```
4. Instala las dependencias necesarias:
    ```
    conda install numpy matplotlib scipy jupyter -y
    ```

    Si alguna falla, puedes usar pip:
    ```
    pip install numpy matplotlib scipy
    ```

5. Ejecuta el script:
    ```
    python semana4_eval_modulacion_fsk.py
    ```
